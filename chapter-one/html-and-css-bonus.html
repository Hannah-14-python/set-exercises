HTML & CSS Quick Reference Guide

HTML Tags

  • <h1> to <h6>: Define HTML headings
  • <p>: Define a paragraph
  • <a>: Create a hyperlink
  • <img>: Insert an image
  • <ul> and <ol>: Create unordered and ordered lists
  • <table>: Create a table
  • <form>: Create an HTML form
  • <input>: Create an input field
  • <button>: Create a button
  • <div>: Define a division or a section in a document
  • <span>: Define a section in a document

CSS Properties

  • color: Set the text color
  • background-color: Set the background color
  • font-family: Set the font family
  • font-size: Set the font size
  • padding: Set the padding around an element
  • margin: Set the margin around an element
  • border: Set the border around an element
  • width: Set the width of an element
  • height: Set the height of an element
